Case Study 01

RecruitFlow — Enterprise Recruitment Platform

Next.js / TypeScript / React / PostgreSQL

The Problem

A growing recruitment agency in Nepal was struggling with fragmented talent management — using spreadsheets, email threads, and WhatsApp groups to track candidate pipelines. This resulted in lost candidates, duplicate outreach, and an average time-to-fill of 45+ days for mid-level positions.

The Solution

I designed and built RecruitFlow — a full-stack recruitment management platform using Next.js with TypeScript for type-safe development. The system features an automated candidate pipeline with drag-and-drop Kanban boards, role-based access control for recruiters and managers, real-time notification systems, and integrated interview scheduling with Google Calendar API. The PostgreSQL database was architected with optimized indexes for fast full-text search across thousands of candidate profiles.

Case Study 02

HamroYatra — Trek Agency Management System

PERN Stack (PostgreSQL / Express / React / Node.js)

The Problem

A Nepal-based travel and trekking agency needed a centralized platform to manage complex multi-day trek itineraries, guide assignments, equipment inventory, and real-time booking management. Their existing paper-based system was causing double-bookings, guide scheduling conflicts, and inventory shortages during peak trekking seasons.

The Solution

I engineered HamroYatra using the full PERN stack — PostgreSQL for relational data integrity across complex trip-guide-equipment relationships, Express.js for a robust RESTful API layer, React for a responsive administrative dashboard, and Node.js for the server runtime. Key features include dynamic itinerary builders with day-by-day customization, real-time availability calendars for guides and equipment, automated booking confirmation emails, and a financial dashboard for revenue tracking and expense categorization.

Case Study 03

Hotel Aarohi — Hospitality Management Platform

PERN Stack / TypeScript / Responsive Design

The Problem

A boutique hotel in Nepal's hospitality sector needed a modern property management system to handle room reservations, guest check-in/check-out workflows, housekeeping task assignments, and revenue reporting. The hotel was losing bookings to competitors who had online booking capabilities while they relied on phone-only reservations.

The Solution

I built a comprehensive hotel management system using the PERN stack with TypeScript for enhanced code reliability. The platform includes a real-time room availability dashboard with visual floor maps, an integrated booking engine that supports both walk-in and online reservations, automated guest communication via email and SMS, housekeeping task management with status tracking, and a financial reporting module with daily revenue summaries, occupancy rate calculations, and seasonal trend analysis. The system was deployed on a VPS with SSL certification and optimized for mobile access by front desk staff.
